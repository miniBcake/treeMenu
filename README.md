# 🌲 TreeMenu

TreeMenu는 부트스트랩(Bootstrap 5) 스타일을 기반으로 한 가볍고 직관적인 트리 메뉴 라이브러리입니다.

### 🚀 Demo
라이브러리의 실제 동작은 아래 데모 페이지에서 확인하실 수 있습니다.  
[데모 페이지 바로가기](https://minibcake.github.io/treeMenu/index.html)

### 🛠 Prerequisites (의존성)
이 라이브러리는 원활한 디자인과 아이콘 표시를 위해 아래의 라이브러리들을 필요로 합니다.
- **Bootstrap** (5.0.2)
- **Bootstrap-icons** (1.11.3)

### ✨ 주요 기능 (Key Features)
저장소에 포함된 예제 파일들을 통해 다음과 같은 기능을 확인하실 수 있습니다.
- **Bootstrap 기반 UI**: 친숙하고 깔끔한 부트스트랩 스타일의 메뉴 구성.
- **JSON 기반 트리 생성**: `jsonTree.html`을 통해 JSON 데이터를 활용한 트리 구조 형성을 지원합니다.
- **지연 로딩(Lazy Loading)**: `lazyLoading.html`을 통해 데이터의 효율적인 로딩 기능을 제공합니다.

### 📁 주요 파일 및 폴더 구조
해당 라이브러리는 다음과 같은 구조로 이루어져 있습니다.

* `js/`: 라이브러리의 로직이 담긴 자바스크립트 폴더.
* `datas/`: 트리에 사용되는 데이터 파일들이 포함된 폴더.
* `index.html`: 프로젝트의 메인 데모 및 사용 예시.
* `jsonTree.html`: JSON 데이터를 활용한 트리 예제.
* `lazyLoading.html`: 지연 로딩 구현 예제.
